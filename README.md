# Adelynn's Jungle Dash 🦁🪙

A fun, kid-friendly coin collection adventure game where players help Adelynn collect coins while avoiding obstacles in a colorful jungle setting!

## 🎮 How to Play

- **Move**: Use arrow keys (← →) or on-screen buttons to move left and right
- **Jump**: Press spacebar, up arrow, or tap the jump button to jump
- **Collect Coins**: Gather golden coins to earn points
- **Avoid Obstacles**: Stay away from logs and other obstacles
- **Survive**: You have 3 lives - don't lose them all!

## 🌟 Features

- **Kid-Friendly Design**: Bright colors, simple controls, and fun animations
- **Mobile Optimized**: Works great on phones and tablets with touch controls
- **Progressive Web App**: Can be installed on devices for offline play
- **Responsive**: Adapts to different screen sizes
- **Sound Effects**: Coin collection and jump sounds
- **Score Tracking**: Keep track of your high scores

## 🚀 Getting Started

1. Open `mainmenu.html` in a web browser
2. Click "Start Game" to begin your adventure
3. Use the controls to help Adelynn collect coins and avoid obstacles
4. Try to beat your high score!

## 📱 Mobile Controls

- **Left/Right Buttons**: Move Adelynn horizontally
- **Jump Button**: Make Adelynn jump
- **Tap Screen**: Alternative way to jump

## 🎨 Game Assets

The game uses custom sprites and audio files:
- Player character sprites
- Coin sprites
- Obstacle sprites (logs, rocks)
- Jungle background
- Sound effects for jumping and coin collection

## 🛠️ Technical Details

- **HTML5 Canvas**: For smooth 2D graphics
- **JavaScript**: Game logic and physics
- **CSS3**: Responsive styling and animations
- **PWA Support**: Offline functionality and app-like experience
- **Service Worker**: Caching for better performance

## 🎯 Game Objectives

- Collect as many coins as possible
- Avoid obstacles to stay alive
- Beat your previous high score
- Have fun exploring the jungle!

## 🌈 Perfect for Kids

- Simple, intuitive controls
- Bright, engaging visuals
- Positive reinforcement (coin collection)
- No violence or scary content
- Educational elements (counting, coordination)
